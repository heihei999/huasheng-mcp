## 公式体系

### 基期量

#### 基期量直接公式法（`da_abx_base_direct_001`）
- A = B / (1 + R)
- X = B - A = B - B/(1+R)
- 也可记为：A = B - X = X/R

#### 隔年前期先求间隔增长率法（`da_abx_base_interval_002`）
- R间 = R1 + R2 + R1×R2
- A隔年 = B / (1 + R间)

#### 前期差值比较法（`da_abx_base_diff_003`）
- A1 - A2 = B1/(1+R1) - B2/(1+R2)

### 现期量

#### 现期量列式与假设增量法（`da_abx_current_001`）
- B = A × (1 + R)
- B = A + X

### 增长率

#### 一般增长率公式法（`da_growth_rate_general_001`）
- R = X / A = (B - A) / A

#### 间隔增长率及逆运用（`da_growth_rate_interval_002`）
- R间 = R1 + R2 + R1×R2
- 逆向：R1 = (R间 - R2)/(1 + R2)

#### 比值增长率公式法（`da_growth_rate_ratio_003`）
- Ra = (Rb - Rc) / (1 + Rc)
- 当只判断升降：Rb > Rc则比值上升，Rb < Rc则比值下降

#### 乘积增长率公式法（`da_growth_rate_product_004`）
- Ra = Rb + Rc + Rb×Rc

### 增长量

#### 增长量公式与方法选择（`da_growth_amount_001`）
- X = B - A = A×R
- X = B×R/(1+R)
- R≤5%且粗略判断时：X≈B×R

#### B-R双大与N倍比较法（`da_growth_amount_compare_002`）
- X = B×R/(1+R)
- B大且R大 => X大；若B大R小，要比较B倍数与R倍数

### 比重

#### 本期比重主体定位法（`da_share_current_001`）
- 本期比重 = 部分 / 整体
- 多部分比重 = 部分和 / 整体

#### 基期比重公式法（`da_share_base_002`）
- 基期比重 = 现期比重 × (1 + R整体)/(1 + R部分)
- 即：a/b = (A/B) × (1+R整体)/(1+R部分)

### 比重变化

#### 比重趋势比较增速法（`da_share_trend_003`）
- R部分 > R整体 => 比重上升
- R部分 < R整体 => 比重下降

#### 比重差公式与秒杀法（`da_share_change_004`）
- 比重差 = 本期比重 - 基期比重
- 近似/公式框架：比重差 ≈ (基期部分/本期整体) × (R部分 - R整体)
- 秒杀：|比重差| < |R部分 - R整体|

#### 三类分式变化辨析表（`da_ratio_share_diff_005`）
- 比值增长率：Ra=(Rb-Rc)/(1+Rc)
- 比重差：A/B-a/b，单位百分点或%
- 比值差：A/B-a/b，单位有实际含义，不可用比重差秒杀

### 平均数

#### 一般平均值‘均前每后’法（`da_average_general_001`）
- 平均数 = 总量 / 份数
- A/B ÷ C/D = AD/BC

#### 年均增长量基期判断法（`da_average_annual_growth_amount_002`）
- 年均增长量 = (末期量 - 基期量) / 年份差n

#### 年均增长率幂次与近似法（`da_average_annual_growth_rate_003`）
- (1 + R)^n = 末期 / 基期
- 末期/基期 ≈ 1 + nR（粗略近似）

### 盐水类

#### 十字交叉求分母之比（`da_salt_cross_001`）
- 若混合值R位于A、B之间，则两部分分母之比 = |R-B| : |A-R|
- 盐水原式类比：(R-B)/(A-R) = 溶液A质量/溶液B质量

#### 混合值不居中定性判断法（`da_salt_qualitative_002`）
- 混合值靠近量较大的部分值
- 增长大小比较：部分1 > 整体 > 部分2

#### 已知2R/3R与量之比反求R法（`da_salt_quantitative_003`）
- R = X/A
- 人均 = 总量/总人数

#### 累计期与单月分段比较法（`da_time_segments_001`）
- 1-n月累计 = 1-(n-1)月累计 + 第n月
- 前段月均 = 1-(n-1)月累计/(n-1)

### 特殊考点

#### 拉动增长率/增量贡献率公式法（`da_ladong_contribution_001`）
- 拉动增长率 = 部分增量 / 整体基期
- 增量贡献率 = 部分增量 / 整体增量

#### 两集合容斥边界判断法（`da_capacity_overlap_001`）
- 至少交集 = a + b - 100%（占比口径）
- 至少交集 = A + B - 总量（数量口径）
- 至多交集 = min(A,B)

## 速算方法

### 截位直除法（`da_truncate_division_001`）
- **适用场景**：式子中主要是除法，尤其是部分/整体、增长率、平均数等需要快速除法；选项差距明显或可通过首两三位判断
- **不适用情况**：选项差距极小且需要精确末位时不能过度截位；式子只有一个除法且分母很小，可直接精算
- **解题步骤**：先看选项差距，不必四个选项都算完整 / 选项差距小，通常截3位；选项首位相同且次位差小于等于首位，保留更多位 / 选项差距大，通常截2位；首位不同可更粗略 / 选项量级不同且接近10倍关系时，截位后要保留量级 / 一步除法通常只截分母；多步混合运算同截分子分母
- **考场速算说明**：先看选项差距；差距大可大胆估，差距小保留更多有效位
- **常见错误**：不看选项差距，截得过粗或过细；截位后忘记量级；只截分子不截分母导致方向偏差
- **来源**：【花生十三】资料分析思维导图.pdf p.4；花生资料分析笔记.pdf p.5

### 415份数法（`da_415_parts_001`）
- **适用场景**：已知现期量B和增长率R，要求增长量X；增长率R靠近常用分数，如1/2、1/3、1/4、1/5等；选项有一定差距，允许估算
- **不适用情况**：R<10%且不接近常用分数时，常用假设分配法更快；选项极接近且R不能稳定转分数时，不宜强用
- **解题步骤**：将增长率R化成相近分数a/b / 把基期、变化量、本期量理解为b:a:(a+b)的份数关系 / 先求一份量，再根据份数求增长量或基期量 / 根据R偏大偏小校正：估大则份数变大、估小则份数变小；增长率为负时要把本期对应份数作相应调整
- **考场速算说明**：先看选项差距；差距大可大胆估，差距小保留更多有效位
- **常见错误**：把R=a/b理解成X/B，而不是X/A；R估大估小后不校正；增长率为负时份数关系照搬正增长
- **来源**：【花生十三】资料分析思维导图.pdf p.4；花生资料分析笔记.pdf p.6

### 假设分配法（`da_assumption_allocation_001`）
- **适用场景**：已知现期量B和增长率R，R较小或不适合415份数法；需要估算基期A与增长量X；选项差距允许一定误差
- **不适用情况**：增长率非常接近常用分数且选项差距大时，415份数法更快；选项极近、要求精确值时，应直除或代入
- **解题步骤**：核心思想：像拆分一样抓大放小，把大数分完，小数误差可忽略 / 确定被分配数和增长率，画出分配树 / 估算基期A和增长量X；最后一步可根据A≈B-R修正 / 根据R大小选择比例：R<20%时X≈BR；R≈25%时A:X≈4:1；R≈33%时A:X≈3:1；R≈50%时A:X≈2:1；R≈66%时A:X≈3:2；R>80%时A和X平分再修正
- **考场速算说明**：先看选项差距；差距大可大胆估，差距小保留更多有效位
- **常见错误**：分配树只分大数不校正小数；把B×R当精确X；增长率为负时符号处理错误
- **来源**：【花生十三】资料分析思维导图.pdf p.4；花生资料分析笔记.pdf p.6

### 尾数-高位叠加-削峰填谷-乘法拆分法（`da_estimation_add_sub_mul_001`）
- **适用场景**：加法选项尾几位不同，可用尾数判断；多个非精确数相加，可从高位叠加；求平均数或多个数接近某基准时可削峰填谷；乘法中一个数接近整百、整千或百分数
- **不适用情况**：选项末位相同且需要高精度时，尾数法不适用；乘法结果需要精确且不能接受估算时，不宜粗拆
- **解题步骤**：尾数法：选项最后有几位不同，就求几位尾数 / 高位叠加：把乘数拆成常见百分数或整十整百后相加 / 削峰填谷：选基准值，用基准+偏离总和/项数求平均 / 乘法拆分：把百分数拆成100%-x%、50%+x%等常见形式
- **考场速算说明**：先看选项差距；差距大可大胆估，差距小保留更多有效位
- **常见错误**：尾数法中借位处理错误；削峰填谷选的基准太远导致计算量增加；乘法拆分后忘记加减小项
- **来源**：【花生十三】资料分析思维导图.pdf p.4；花生资料分析笔记.pdf p.4,5

### 常用分数大小比较法（`da_fraction_compare_difference_001`）
- **适用场景**：需要比较两个分数大小；分子分母同大同小或一大一小，直接除法较慢；题目属于比重、平均数、比值大小比较
- **不适用情况**：分子分母一大一小时可直接看：分子大分母小者大，不必差分；选项或分数差距很大时，可直接估算或截位直除
- **解题步骤**：先判断分子、分母相对大小：分子大且分母小则该分数更大；分子小且分母大则更小 / 若分子分母同时变大或同时变小，优先纵向直除或横向比较倍数 / 只在无法直接判断时再截位计算关键分数
- **考场速算说明**：先看选项差距；差距大可大胆估，差距小保留更多有效位
- **常见错误**：PDF中明确标题是“常用分数比较/判断分数大小”，未确认其标题为“差分法”；分子、分母同时变大时不能仅看分子大就判更大；不要把用于比较的估算过程当成精确计算结果
- **来源**：【花生十三】资料分析思维导图.pdf p.6；花生资料分析笔记.pdf p.5

## 各题型解题流程

### 基础知识

#### 资料分析基础术语与审题四步（`da_base_terms_001`）
- **适用场景**：题干出现同比、环比、倍数、翻番、顺差、逆差、产业增加值、人口自然增长率等基础统计术语；题目要求先判断时间、主体、单位、图例或数据口径
- **题干触发词**：统计术语；同比；环比；倍数；翻番；顺差逆差；人口自然增长率；审题
- **不适用情况**：已经明确是某一公式型题，且无需术语判断时不作为主方法；纯计算题不需要反复解释术语，只保留审题校验
- **必需输入**：题干原文；材料时间范围；统计指标名称；单位/图例信息
- **解题步骤**：先判断问的是基期还是现期，特别注意同比与环比的参照期 / 圈出主体和关键词，确认增长率、增长量、比重、平均数等指标类别 / 检查单位是否一致，优先找不同单位与选项单位 / 图表题先看图例、起止年份、是否有合计或总计行 / 遇到饼状图按12点钟方向顺时针读比例
- **考场速算说明**：作为审题和校验规则使用。 按选项差距决定估算、截位或精算
- **输出约束**：输出前必须核对时间、主体、单位；若为选择题，优先给出最接近选项
- **常见错误**：把产业增加值误当成增长量；把同比参照期和环比参照期混淆；忽略单位或图例导致读错数据；人口自然增长率单位为千分数
- **来源**：花生资料分析笔记.pdf p.3

### 基期量

#### 基期量直接公式法（`da_abx_base_direct_001`）
- **适用场景**：题干给出现期量B和同比增长率R，问上年、去年、前期、基期量；题干出现‘比上年增长/下降R%’，问题问上一时期的量
- **题干触发词**：ABRX；基期量；前期量；现期量；增长率
- **不适用情况**：题目要求隔年前期时，应先处理间隔增长率；题目只比较大小且选项差距很大时，可用估算或415份数法替代精算
- **必需输入**：现期量B；增长率R；目标基期时间；选项单位
- **解题步骤**：识别现期量B与增长率R / 增长为正时用A=B/(1+R)，下降为负时用A=B/(1-R的绝对值) / 若R接近常用分数，转化为份数估算 / 结合选项差距决定精算、直除或估算
- **考场速算说明**：先识别ABRX/分式结构，再套用对应公式；能估则估，必要时直除。 按选项差距决定估算、截位或精算
- **输出约束**：输出前必须核对时间、主体、单位；若为选择题，优先给出最接近选项；结果通常输出为百分数%；不得把百分点变化直接当作增长率；结果单位应与材料原量单位一致；下降率参与分母时要保留负向关系
- **常见错误**：把现期量B误当基期量A；增长率下降时分母应小于1，不可机械写1+R正数；选项差距大时过度精算浪费时间
- **来源**：【花生十三】资料分析思维导图.pdf p.1；花生资料分析笔记.pdf p.7

#### 隔年前期先求间隔增长率法（`da_abx_base_interval_002`）
- **适用场景**：题干要求求隔年、两年前、与2014年相比的前期量；题干给出连续两期增长率或增长率相对前一年提高/下降若干百分点
- **题干触发词**：ABRX；基期量；间隔增长率；隔年前期
- **不适用情况**：若直接给出目标前期值，不必计算间隔增长率；若只问两期增长率本身，调用间隔增长率卡片
- **必需输入**：现期量B；相邻两期增长率R1/R2或等价信息；目标隔年前期时间；选项差距
- **解题步骤**：先根据相邻两期增长率求总间隔增长率R间 / 再把现期量B代入A=B/(1+R间) / 若R间接近常用分数，可用415份数法或假设分配法估算
- **考场速算说明**：先识别ABRX/分式结构，再套用对应公式；能估则估，必要时直除。 按选项差距决定估算、截位或精算
- **输出约束**：输出前必须核对时间、主体、单位；若为选择题，优先给出最接近选项；结果单位应与材料原量单位一致；下降率参与分母时要保留负向关系
- **常见错误**：遗漏R1×R2项；把‘增速下降百分点’直接当负增长率；年份方向反了，导致用错基期
- **来源**：【花生十三】资料分析思维导图.pdf p.1；花生资料分析笔记.pdf p.7,8

#### 前期差值比较法（`da_abx_base_diff_003`）
- **适用场景**：题干问两个对象在上年、去年、前期相差多少；题干给出现期量与增长率，需要比较A1-A2
- **题干触发词**：ABRX；前期差值；基期差；比较
- **不适用情况**：只问单个前期量时优先用基期量直接公式法；选项为趋势判断时不必求精确差值
- **必需输入**：两个对象的现期量B1/B2；两个对象的增长率R1/R2；需要比较的前期差值方向或数值
- **解题步骤**：分别识别两个对象的现期量B1、B2及增长率R1、R2 / 分别估算前期A1、A2 / 计算A1-A2并根据选项差距确定精度
- **考场速算说明**：先识别ABRX/分式结构，再套用对应公式；能估则估，必要时直除。 按选项差距决定估算、截位或精算
- **输出约束**：输出前必须核对时间、主体、单位；若为选择题，优先给出最接近选项；结果单位应与材料原量单位一致；下降率参与分母时要保留负向关系
- **常见错误**：直接用现期差替代前期差；两个对象增长率不同却只用一个折扣；差值正负方向写反
- **来源**：【花生十三】资料分析思维导图.pdf p.1；花生资料分析笔记.pdf p.7

### 现期量

#### 现期量列式与假设增量法（`da_abx_current_001`）
- **适用场景**：题干给基期量A及增长率R，问现期量B；题干给出未来若干年保持相同增长量或增长率，问某年现期量；题干要求‘求当年、本期、今年、未来第n年’
- **题干触发词**：ABRX；现期量；本期量；增长量；增长率
- **不适用情况**：只问增长量时优先调用增长量卡片；增长率极小且选项差距大时可直接估算，不必完整列式
- **必需输入**：基期量A或当前量；增长率R或增长量X；保持增量/保持增速条件；目标年份或目标阈值
- **解题步骤**：若已知A和R，直接列B=A×(1+R) / 若已知A和X，直接列B=A+X / 若未来保持增长量不变，先设X并列出‘当年量+n年增长量X’的不等式或方程 / 若未来保持增速不变，逐年乘(1+R)，通常2到3次即可判断选项
- **考场速算说明**：先识别ABRX/分式结构，再套用对应公式；能估则估，必要时直除。 按选项差距决定估算、截位或精算
- **输出约束**：输出前必须核对时间、主体、单位；若为选择题，优先给出最接近选项；结果单位应与原始数量单位一致；减少量可输出正值并说明方向；结果单位应与材料原量单位一致；下降率参与分母时要保留负向关系
- **常见错误**：保持增长量与保持增长率混淆；未来多期推算时把基期年份数错；列不等式时忽略‘超过/达到’的边界
- **来源**：【花生十三】资料分析思维导图.pdf p.1；花生资料分析笔记.pdf p.7,8

#### 间隔增长率及逆运用（`da_growth_rate_interval_002`）
- **适用场景**：题干给连续两期增长率，问隔年或跨期增长率；题干给本期较上期、上期较再上一期的增速，要求跨期比较；需要由R间和其中一个R反推另一个R

- **常见错误**：只加不乘，尤其当选项接近时容易错；把‘增长率下降0.8个百分点’理解成R2=-0.8%；倒推时忽略分母1+R

#### 比值增长率公式法（`da_growth_rate_ratio_003`）

- **不适用情况**：题干出现‘占...比重’且问百分点变化，应调用比重差卡片；分子是分母一部分且问比重变化，不属于比值增长率

- **解题步骤**：判断比值指标A=B/C，分子增长率为Rb，分母增长率为Rc / 代入(Rb-Rc)/(1+Rc) / 若只判断正负，比较Rb与Rc即可 / 若问平均数增长率，记住‘均前每后’：平均、每后面的量通常作分母

- **常见错误**：把比值增长率误当比重差，选百分点选项；忘记分母1+Rc；平均数题中分子分母倒置

#### 乘积增长率公式法（`da_growth_rate_product_004`）
- **适用场景**：题干中的目标量可拆成B×C；题干给出一个总量增长率和占比变化/占比增长率，要求目标量增长率；资料中存在‘整体×占比=部分’关系
- **题干触发词**：乘积增长率；A=B×C；占比；增长率
- **不适用情况**：B×C只是单位换算且无增长率时不作为增长率题处理；若C是比重差的百分点变化而不是增长率，需要先转换或改用比重类方法

- **解题步骤**：确认目标量A=B×C是否有实际含义或表示占比关系 / 识别B、C各自增长率Rb、Rc / 代入Ra=Rb+Rc+RbRc；若增长率较小可先估算Rb+Rc，再校验乘积项

- **常见错误**：漏掉Rb×Rc；误把占比提高的百分点当成占比增长率；未判断B×C是否真能表达目标量

### 增长量

- **解题步骤**：看分子分母单位：相同且分子是整体一部分，多为比重/比重差 / 看选项单位：%常见于比值增长率；百分点常见于比重差 / 看问法：‘平均...增长率’多为比值增长率；‘占...比重’一定优先归比重类 / 若分子不是分母一部分但形式为A/B-a/b，则是比值差，可套公式但不可秒杀

- **必需输入**：集合A占比/数量；集合B占比/数量；总体数量或100%口径；问至少还是至多
- **解题步骤**：若两个占比和超过100%，则一定存在交集 / 问至多时，交集最大为较小占比 / 问至少时，交集最小为a+b-100%或a+b-总量 / 若有具体总量，用a+b-总量求至少交集人数
- **考场速算说明**：先识别特殊问法，再调用专门公式；不混入普通增长率或普通比重。 按选项差距决定估算、截位或精算
- **输出约束**：输出前必须核对时间、主体、单位；若为选择题，优先给出最接近选项；按题干要求输出百分数、百分点或数量边界；不得混用普通比重差秒杀规则
- **常见错误**：问至多却用a+b-100%；问至少却直接取较小集合；两个集合不在同一总体下强行容斥
- **来源**：花生资料分析笔记.pdf p.14

### 易错点

#### 资料分析常见陷阱清单（`da_common_pitfalls_001`）
- **适用场景**：任意资料分析题的最终校验阶段；题干时间、单位、主体、问法存在多层限定
- **题干触发词**：易错点；陷阱；审题；单位；时间；百分点
- **不适用情况**：无，不作为主解法，只作为校验清单
- **必需输入**：题干原文；材料原表/图；选项单位；时间和主体口径
- **解题步骤**：核对时间：现期/基期、同比/环比、起止年份、月份累计 / 核对主体：部分/整体、分子/分母、问的是谁 / 核对单位：亿元/万元、万人/人、百分数/百分点 / 核对问法：增长率、增长量、比重、比重差、平均数增长率是否混淆 / 核对图表：图例、合计总计、第一年是否能求增量 / 核对选项：选项差距决定截位精度
- **考场速算说明**：作为审题和校验规则使用。 按选项差距决定估算、截位或精算
- **输出约束**：作为最终校验清单输出；不得替代主方法求解；必须标记高风险口径：时间、主体、单位、图例、百分数/百分点
- **常见错误**：百分点与百分数混淆；同比与环比混淆；现期量与基期量混淆；增长量和增长率混淆；平均数增长率与比重差混淆；选项差距小却粗算
- **来源**：【花生十三】资料分析思维导图.pdf p.2,3,6；花生资料分析笔记.pdf p.3,9,12,13

## 易错点

- `da_base_terms_001`：把产业增加值误当成增长量；把同比参照期和环比参照期混淆；忽略单位或图例导致读错数据
- `da_abx_base_direct_001`：把现期量B误当基期量A；增长率下降时分母应小于1，不可机械写1+R正数；选项差距大时过度精算浪费时间
- `da_abx_base_interval_002`：遗漏R1×R2项；把‘增速下降百分点’直接当负增长率；年份方向反了，导致用错基期
- `da_abx_base_diff_003`：直接用现期差替代前期差；两个对象增长率不同却只用一个折扣；差值正负方向写反
- `da_abx_current_001`：保持增长量与保持增长率混淆；未来多期推算时把基期年份数错；列不等式时忽略‘超过/达到’的边界
- `da_growth_rate_general_001`：把增长量X除以现期量B；把百分点变化当成增长率；忽视单位导致X和A口径不同
- `da_growth_rate_interval_002`：只加不乘，尤其当选项接近时容易错；把‘增长率下降0.8个百分点’理解成R2=-0.8%；倒推时忽略分母1+R
- `da_growth_rate_ratio_003`：把比值增长率误当比重差，选百分点选项；忘记分母1+Rc；平均数题中分子分母倒置
- `da_growth_rate_product_004`：漏掉Rb×Rc；误把占比提高的百分点当成占比增长率；未判断B×C是否真能表达目标量
- `da_growth_amount_001`：用B×R作为精确增长量，忽略分母1+R；题干问减少量却把负号带入选项；增长率很大时仍用近似导致误差过大
- `da_growth_amount_compare_002`：只看增长率高低，不看基数大小；两个对象一大一小时直接下结论；没有排除明显小项，耗时计算所有对象
- `da_share_current_001`：分子分母反了；把题干中的中间量当整体；多个部分未合并就直接除
- `da_share_base_002`：把R整体和R部分的位置写反；没有先化出现期比重；把百分点提高当成R部分
- `da_share_trend_003`：把比重上升误认为增长量更大；没有确认部分属于整体；只看现期比重大小判断趋势
- `da_share_change_004`：把百分点写成百分比增长率；忘记先判方向；比值差题错误使用‘比重差小于增速差’秒杀
- `da_ratio_share_diff_005`：看到分式就全部套比重差；把平均数增长率误作比重差；比值差使用比重差秒杀
- `da_average_general_001`：分母找错，尤其‘平均每’结构；时间范围少算或多算月份；除法未约分导致计算量变大
- `da_average_annual_growth_amount_002`：直接把题面起始年份当基期，忽略前一年；年份差少除或多除1；图表第一年是否可直接使用没有判断
- `da_average_annual_growth_rate_003`：把年均增长率当成(末期-基期)/年份差；n取错；选项接近时过度使用线性近似
- `da_compare_ratio_growth_001`：只看分子大小，不看分母变化；图表中混淆柱状和折线指标；比较年份时忽略第一年的增长量不可直接求
- `da_chart_lookup_001`：把合计行当普通分项参与排序；忽略单位换算；第一年缺少前期却强算增长量
- `da_truncate_division_001`：不看选项差距，截得过粗或过细；截位后忘记量级；只截分子不截分母导致方向偏差
- `da_415_parts_001`：把R=a/b理解成X/B，而不是X/A；R估大估小后不校正；增长率为负时份数关系照搬正增长
- `da_assumption_allocation_001`：分配树只分大数不校正小数；把B×R当精确X；增长率为负时符号处理错误
- `da_estimation_add_sub_mul_001`：尾数法中借位处理错误；削峰填谷选的基准太远导致计算量增加；乘法拆分后忘记加减小项
- `da_fraction_compare_difference_001`：PDF中明确标题是“常用分数比较/判断分数大小”，未确认其标题为“差分法”；分子、分母同时变大时不能仅看分子大就判更大；不要把用于比较的估算过程当成精确计算结果
- `da_salt_cross_001`：把十字交叉结果当分子之比；整体值不在两个部分值之间仍硬套；R代入增长率时忘记得到的是基期量之比
- `da_salt_qualitative_002`：误以为混合值一定在两个数正中间；不看量的大小，只看数值距离；忽略题目问的是增长率范围还是占比
- `da_salt_quantitative_003`：把人均相关题直接当人数之比；忘记增长率代入十字交叉时求出的是基期量之比；正负增长率在同一个线段中位置判断错误
- `da_time_segments_001`：把累计值当单月值；1-7月和7月重复计算；比较平均时分母月份数错
- `da_ladong_contribution_001`：把整体现期当整体基期；把部分现期量当部分增量；拉动增长率和贡献率分母混淆
- `da_capacity_overlap_001`：问至多却用a+b-100%；问至少却直接取较小集合；两个集合不在同一总体下强行容斥
- `da_common_pitfalls_001`：百分点与百分数混淆；同比与环比混淆；现期量与基期量混淆

## 考场优先级

1. 先调用 `da_base_terms_001` 做时间、主体、单位、图例和术语预检。
2. 再按题干触发词调用核心公式卡：ABRX、比重、平均数、比较类等。
3. 选项差距允许时，调用速算卡：截位直除、415份数法、假设分配法、分数比较。
4. 题干出现盐水结构、拉动/贡献率、容斥时，优先调用特殊考点卡。
5. 输出前统一调用 `da_common_pitfalls_001` 做最终口径校验。
