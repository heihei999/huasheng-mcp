# 判断推理-图形推理方法库
## 题型总览

图推先按命题形式和图形组成进入不同分支。命题形式包括一组图、两组图、九宫格、分组分类和空间类；组成判断包括相同、相似、不同和特殊图形。总入口卡片为 `gr_total_flow_001`。
## 公式/口诀体系

- 十五字口诀：屈臣氏整风，点线面角素，直接想位移。
- 组成相同：优先看平移、旋转、翻转。
- 组成相似：优先看遍历、加减同异、黑白运算。
- 组成不同：优先看属性，再看数量。
- 空间重构：相对面排除 -> 公共边/公共点 -> 时针法 -> 选项验证。
## 总观察顺序

1. 识别命题形式：一组图、两组图、九宫格、分组分类、空间类。
2. 判断图形组成：相同/相似/不同/特殊。
3. 相同看位置，相似看样式，不同看属性与数量，特殊看黑白块/汉字/数字/字母。
4. 数量题必须先定义计数对象：点、线、角、面、素、笔画。
5. 空间题必须先排除后验证，不能凭立体直觉。
6. 所有题最终必须执行选项验证流程。
## 图推总流程
### 图推总流程卡片（`gr_total_flow_001`）
- 适用场景：遇到图形推理题；需要判断先看什么规律；题干没有明显单一特征时
- 题干触发词/图形特征：总流程；观察顺序；图形组成；十五字口诀；优先调用
- 不适用情况：已经由明显特殊题型直接锁定时可进入对应专项卡片，但仍需回到总流程验证
- 解题步骤：
  1. 先识别命题形式：一组图、两组图、九宫格、分组分类、空间类。
  2. 再判断图形组成：元素组成相同优先位置规律；相似优先样式规律；不同优先属性规律、数量规律；特殊图形优先黑白块、汉字、数字、字母等专项。
  3. 按十五字口诀排查：屈臣氏整风，点线面角素，直接想位移。
  4. 有明显特征先做“特征图优先”，没有明显特征再从属性到数量再到位置/样式逐级验证。
  5. 最终必须回到选项验证：候选选项是否同时满足已发现规律，若只满足局部规律不得直接选。
- 检查顺序：命题形式 -> 元素组成相同/相似/不同/特殊 -> 属性：曲直/对称/开闭 -> 数量：点/线/面/角/素/笔画 -> 位置：方向/移动/旋转/翻转 -> 样式：遍历/加减同异/黑白运算 -> 空间：相对面/公共边/截面/三视图/拼合 -> 选项验证
- 考场速算/操作说明：ordered_visual_screening，优先级 1。
- 常见错误：只看某一个图形特征就选项；元素组成未判断清楚就直接数点线面；空间题不做选项验证
- 来源：花生图形推理笔记.pdf p.3,4；【判断推理】图形推理.pdf p.1

### 图推分三轮处理策略（`gr_exam_pacing_001`）
- 适用场景：一组图推题数量较多；短时间看不出规律；需要避免卡题
- 题干触发词/图形特征：考场策略；时间分配；放弃策略
- 不适用情况：单题已经明显可解时不需要拖到下一轮
- 解题步骤：
  1. 第一轮用3-4分钟先解决3-5道简单题。
  2. 判断模块做完后用2-3分钟回看剩余题。
  3. 涂卡阶段再利用直觉回看难题，不因个别图推拖慢整体速度。
- 检查顺序：先做明显题 -> 回看剩余题 -> 最后涂卡阶段再补看
- 考场速算/操作说明：time_boxed_solving，优先级 5。
- 常见错误：连续盯一题超过合理时间；因图推影响整个判断模块节奏
- 来源：【判断推理】图形推理.pdf p.1

### 命题形式与读取顺序卡（`gr_question_form_001`）
- 适用场景：题干呈现一行或两行图形；九宫格中缺一格；要求分组分类；出现立体展开/截面/三视图
- 题干触发词/图形特征：一组图；两组图；九宫格；分组分类；立体类；读取顺序
- 不适用情况：不要在未确定读取顺序时直接套数量递推
- 解题步骤：
  1. 一组图：从左往右观察，必要时看间隔或奇偶项。
  2. 两组图：第一组找规律，第二组对应套用。
  3. 九宫格：横行、竖列、米字或中心特殊均可尝试。
  4. 分组分类：先找共性，再找差异，优先考虑三三分组。
  5. 立体类：从展开图、截面图、三视图、立体拼合分类进入专项。
- 检查顺序：先判断一组/两组/九宫格/分组/空间 -> 再确定读取方向 -> 最后进入对应规律卡
- 考场速算/操作说明：form_first_routing，优先级 2。
- 常见错误：九宫格只横看不竖看；分组分类只看相似不看差异；立体题用平面位置规律硬解
- 来源：花生图形推理笔记.pdf p.3；【判断推理】图形推理.pdf p.1

### 拆分思维与对比思维卡（`gr_split_compare_thinking_001`）
- 适用场景：整体规律不明显；图形由上下、内外、左右、横线、黑白、曲直等部分构成；分组分类题需要找相同与不同
- 题干触发词/图形特征：拆分思维；对比思维；局部观察；分组分类
- 不适用情况：整体规律非常明确时不必过度拆分；拆分后各部分无稳定对应关系时回到总流程
- 解题步骤：
  1. 位置拆分可看上下、内外、左右或横线分开规律。
  2. 样式拆分可看黑白、曲直分开规律。
  3. 对比图形看不变部分，对比题干看相邻变化。
  4. 分组分类先找共性再找差异。
- 检查顺序：上下/内外/左右/横线 -> 黑白/曲直 -> 不变部分 -> 相邻变化 -> 选项验证
- 考场速算/操作说明：decompose_then_compare，优先级 25。
- 常见错误：拆分维度过多导致自造规律；只看某个局部不验证全图
- 来源：花生图形推理笔记.pdf p.4

### 十五字口诀调用卡（`gr_mnemonic_15_001`）
- 适用场景：需要快速遍历图推考点；没有明显切入点；AI解题器需要统一搜索顺序
- 题干触发词/图形特征：十五字口诀；屈臣氏整风；点线面角素；直接想位移
- 不适用情况：已有明确专项触发时可先用专项，但答题前仍回到口诀检验是否遗漏
- 解题步骤：
  1. 屈：曲线直线；臣：对称；氏：相同相似图形；整：整体部分；风：封闭开放。
  2. 点线面角素用于数量观察。
  3. 直接想位移用于位置观察：垂直/平行、连接方式、方向、位置、平移、旋转。
  4. 按口诀生成候选规律，不得跳步猜答案。
- 检查顺序：屈 -> 臣 -> 氏 -> 整 -> 风 -> 点线面角素 -> 直接想位移 -> 选项验证
- 考场速算/操作说明：mnemonic_guided_screening，优先级 3。
- 常见错误：背口诀但不判断组成；只看数量不看属性；位移类忘记先确认组成相同
- 来源：【判断推理】图形推理.pdf p.1

## 组成判断
### 组成相同优先看位置规律（`gr_composition_same_001`）
- 适用场景：每幅图的元素种类和数量基本相同；图形只是位置、方向或角度变化；图形轮廓完全一致或仅摆放不同
- 题干触发词/图形特征：组成相同；位置规律；平移；旋转；翻转
- 不适用情况：元素数量或种类明显变化时不要优先看位置；出现加减叠加痕迹时转样式规律
- 解题步骤：
  1. 核对每图元素种类与数量是否一致。
  2. 若一致，优先看平移路径、旋转角度、翻转轴向。
  3. 在九宫格中按行列分别验证位置变化是否一致。
  4. 用选项逐一模拟下一步位移/旋转/翻转。
- 检查顺序：元素种类 -> 元素数量 -> 移动路径 -> 方向/角度 -> 选项模拟
- 考场速算/操作说明：position_first，优先级 3。
- 常见错误：把相似图误判为相同图；只看整体，不看内部小元素的移动
- 来源：花生图形推理笔记.pdf p.4,5；【判断推理】图形推理.pdf p.1

### 组成相似优先看样式规律（`gr_composition_similar_001`）
- 适用场景：图形轮廓或元素有明显相似关系；局部线条、阴影、空白、黑白块重复出现；相邻图之间存在叠加、去同、去异迹象
- 题干触发词/图形特征：组成相似；遍历；加减同异；黑白运算；样式规律
- 不适用情况：元素完全一致且只变位置时先看位置；元素差异过大且无叠加关系时先看属性/数量
- 解题步骤：
  1. 观察相同元素是否在行列中遍历出现。
  2. 检查相邻图是否存在加、减、去同、去异。
  3. 若是黑白格或黑白块，按同位置运算验证。
  4. 验证选项是否能由相同样式规则推出。
- 检查顺序：是否遍历 -> 是否加减同异 -> 是否黑白运算 -> 是否选项唯一
- 考场速算/操作说明：style_rule_first，优先级 4。
- 常见错误：只凭外观相似而忽略行列遍历；未按同一位置比较黑白块
- 来源：花生图形推理笔记.pdf p.4,6,17；【判断推理】图形推理.pdf p.1

### 组成不同优先看属性与数量（`gr_composition_different_001`）
- 适用场景：各图元素差异较大；没有明显相同元素可移动或叠加；每图整体特征可归类
- 题干触发词/图形特征：组成不同；属性规律；数量规律；曲直；对称；开闭；点线面角素
- 不适用情况：存在明显相同元素移动时不要先数数量；有黑白块/汉字/数字/字母等特殊题型时先走专项
- 解题步骤：
  1. 先看属性：曲直性、对称性、开闭性。
  2. 属性无规律再看数量：点、线、角、面、素、笔画。
  3. 出现明显提示图形时优先使用提示考点，如T点、出头、田字、五角星等。
  4. 用候选规律反查题干所有图形与选项。
- 检查顺序：曲直 -> 对称 -> 开闭 -> 点 -> 线/笔画 -> 角 -> 面 -> 素 -> 选项验证
- 考场速算/操作说明：attribute_then_quantity，优先级 6。
- 常见错误：没有触发点就盲目数复杂数量；只数前两个图后直接选
- 来源：花生图形推理笔记.pdf p.4,7,8,9,10,11,12,13,14；【判断推理】图形推理.pdf p.1

### 特殊图形优先专项路由（`gr_special_shape_routing_001`）
- 适用场景：题干主体为黑白块、汉字、数字、字母；图形具有强符号属性；常规属性数量难以解释但符号本身有规律
- 题干触发词/图形特征：黑白块；汉字；数字；字母；专项题型
- 不适用情况：图形不是符号类且元素组成清晰时不强行走专项
- 解题步骤：
  1. 识别是否为黑白块、汉字、数字或字母。
  2. 黑白块先看对称/面积/数量/连接/运算。
  3. 汉字先看结构、样式、属性、数量、特殊读音或笔顺。
  4. 数字和字母先看对称、曲直、开闭、面数，再看本身大小、顺序或运算。
- 检查顺序：黑白块 -> 汉字 -> 数字 -> 字母 -> 专项规律 -> 返回总流程验证
- 考场速算/操作说明：special_type_routing，优先级 7。
- 常见错误：把数字/字母只当普通图形，忽略本身顺序；黑白块不按位置比较
- 来源：花生图形推理笔记.pdf p.2,17,18,19,20；【判断推理】图形推理.pdf p.1

## 位置规律
### 平移与移动路径法（`gr_position_translate_001`）
- 适用场景：元素组成相同；某个元素在不同位置出现；题干呈宫格或有明确路径
- 题干触发词/图形特征：平移；移动；路径；宫格；方向
- 不适用情况：元素数量变化明显；相邻图存在叠加/消失/新增时不优先看平移
- 解题步骤：
  1. 确定移动主体：整体轮廓、小元素、空白、顶点/交点或线条。
  2. 确定移动路径：直线、顺/逆时针、横向、纵向、内外圈。
  3. 记录每一步移动格数或位置变化。
  4. 在选项中模拟下一步位置。
- 检查顺序：找主体 -> 找路径 -> 找步长 -> 验证选项
- 考场速算/操作说明：simulate_translation，优先级 12。
- 常见错误：移动主体选错；九宫格中只看横向不看纵向；忽略空白或交点也可能是移动主体
- 来源：花生图形推理笔记.pdf p.5；【判断推理】图形推理.pdf p.1

### 旋转角度与方向法（`gr_position_rotation_001`）
- 适用场景：元素组成相同；图形整体或局部方向连续变化；每幅图只是转向变化
- 题干触发词/图形特征：旋转；顺时针；逆时针；角度
- 不适用情况：图形存在镜像翻转时不要误判旋转；角度不稳定且元素位置改变复杂时先结合平移
- 解题步骤：
  1. 确认旋转主体是整体还是局部。
  2. 判断方向：顺时针或逆时针。
  3. 判断角度：恒定、递增或递减。
  4. 将候选选项旋转回题干位置进行比对。
- 检查顺序：主体 -> 中心 -> 方向 -> 角度 -> 镜像排除 -> 选项验证
- 考场速算/操作说明：rotation_sequence，优先级 13。
- 常见错误：把翻转后的镜像误当旋转；未判断旋转中心
- 来源：花生图形推理笔记.pdf p.5；【判断推理】图形推理.pdf p.1

### 翻转轴与镜像识别法（`gr_position_flip_001`）
- 适用场景：元素组成相同；相邻图呈镜像关系；横向或纵向左右互换/上下互换明显
- 题干触发词/图形特征：翻转；横轴对称；竖轴对称；中心对称；镜像
- 不适用情况：图形只是角度转动且非镜像时不要看翻转；内部标识不支持镜像时须排除
- 解题步骤：
  1. 判断两图是否横轴对称、竖轴对称或中心对称。
  2. 横轴对称对应上下翻，竖轴对称对应左右翻。
  3. 中心对称通常可视为旋转180度。
  4. 用不对称标记点验证是否真的翻转。
- 检查顺序：横轴 -> 竖轴 -> 中心 -> 内部标记 -> 选项镜像验证
- 考场速算/操作说明：mirror_check，优先级 14。
- 常见错误：对称图形本身看不出翻转，需找不对称标记；把180度旋转和上下/左右翻混用
- 来源：花生图形推理笔记.pdf p.5；【判断推理】图形推理.pdf p.1

## 样式规律
### 样式/数量/位置遍历法（`gr_style_traversal_001`）
- 适用场景：相同元素重复出现；每行每列元素种类或数量可能相同；九宫格或两组图中有元素循环
- 题干触发词/图形特征：遍历；样式规律；重复出现；行列次数
- 不适用情况：图形组成完全不同且无重复元素时不要优先遍历；有明显移动主体时先看位置
- 解题步骤：
  1. 列出每行/每列出现过的元素、样式或位置。
  2. 检查缺失项是否为某一类元素、某一数量或某一位置。
  3. 对九宫格按行列分别尝试。
  4. 选项必须补足缺失遍历项且不破坏已有次数。
- 检查顺序：列元素清单 -> 看每行 -> 看每列 -> 找缺失 -> 选项补足
- 考场速算/操作说明：set_completion，优先级 20。
- 常见错误：只看样式不看位置；把重复次数看错；没有逐行逐列核对
- 来源：花生图形推理笔记.pdf p.6；【判断推理】图形推理.pdf p.1

### 加减同异与去同去异法（`gr_style_add_sub_same_diff_001`）
- 适用场景：相邻图有线条或元素叠加关系；两图合成第三图；相同部分可能保留或消失
- 题干触发词/图形特征：加减同异；去同存异；去异存同；叠加
- 不适用情况：元素没有对应位置关系时不要强行做加减；黑白块题应先按黑白运算专项核对
- 解题步骤：
  1. 对齐相邻图形。
  2. 判断运算方式：直接相加、相减、去同存异、去异存同。
  3. 在每个相同位置或相同线段上验证运算。
  4. 用选项替入，看是否能完成同样运算。
- 检查顺序：对齐 -> 找相同 -> 找不同 -> 确定保留/删除 -> 选项代入
- 考场速算/操作说明：boolean_shape_operation，优先级 21。
- 常见错误：未对齐图形就运算；只验证一处线条；把位置移动误认为叠加
- 来源：花生图形推理笔记.pdf p.6；【判断推理】图形推理.pdf p.1

### 黑白块同位置运算法（`gr_black_white_operation_001`）
- 适用场景：图形由黑白格/黑白块组成；每幅图格子结构一致；黑白数量或位置随行列变化
- 题干触发词/图形特征：黑白运算；黑白块；同位置比较；加减同异
- 不适用情况：格子结构不一致或无同位置可比时不优先黑白运算；黑块数量完全相同且整体平移明显时先看平移
- 解题步骤：
  1. 先确认所有图格位一致。
  2. 按同一位置比较黑白状态。
  3. 尝试黑+黑、黑+白、白+黑、白+白对应结果。
  4. 用一整行或一整列建立运算表，再用另一行/列验证。
  5. 选项必须在所有格位上满足同一运算。
- 检查顺序：同格位 -> 建运算表 -> 用另一组验证 -> 代入选项
- 考场速算/操作说明：cellwise_boolean，优先级 22。
- 常见错误：只比较黑块数量，不逐格运算；行列运算方向混乱；用不同运算解释不同格位
- 来源：花生图形推理笔记.pdf p.6,17；【判断推理】图形推理.pdf p.1

## 属性规律
### 对称性与对称轴法（`gr_property_symmetry_001`）
- 适用场景：图形呈规整形状；出现Z/S/风车/太极/平行四边形等中心对称提示；正三角形、田字、五角星、正六边形等提示对称轴
- 题干触发词/图形特征：对称性；轴对称；中心对称；对称轴数量；对称轴方向
- 不适用情况：图形组成相同且明显移动时不要先看对称；对称性无稳定规律时转数量或位置
- 解题步骤：
  1. 判断是否轴对称、中心对称或二者兼具。
  2. 统计对称轴数量是否恒定、递增递减或运算。
  3. 判断对称轴方向是否不变或依次变化。
  4. 必要时观察对称轴是否经过点、线、面或与图形关系平行/垂直。
  5. 用选项验证对称类型和轴数方向。
- 检查顺序：轴对称 -> 中心对称 -> 轴数 -> 轴方向 -> 轴与图形关系 -> 选项验证
- 考场速算/操作说明：symmetry_classification，优先级 30。
- 常见错误：把近似对称当严格对称；只看轴数不看方向；忽略中心对称
- 来源：花生图形推理笔记.pdf p.7；【判断推理】图形推理.pdf p.1

### 开闭性判断法（`gr_property_open_closed_001`）
- 适用场景：图形有缺口或封闭空间；出现0/4/6/8/9等数字、字母或类似闭合图形；组成元素不同且曲直无规律
- 题干触发词/图形特征：开闭性；开放图形；封闭图形；半开半闭
- 不适用情况：图形仅开闭变化不稳定且有更明显对称/曲直规律时后置；封闭面数量变化明显时转面数量
- 解题步骤：
  1. 判断每图是开放、封闭还是半开半闭。
  2. 看开闭状态是否分组、交替或递推。
  3. 若出现数字/字母，按其封闭面数量或开闭属性核对。
  4. 选项开闭状态必须与题干序列一致。
- 检查顺序：开放 -> 封闭 -> 半开半闭 -> 交替/分组 -> 选项验证
- 考场速算/操作说明：open_closed_classification，优先级 31。
- 常见错误：把有阴影区域误当封闭空间；忽略图形缺口；开闭性和封闭面数量混淆
- 来源：花生图形推理笔记.pdf p.8,19,20；【判断推理】图形推理.pdf p.1

### 曲直性判断法（`gr_property_curve_straight_001`）
- 适用场景：图形由明显曲线和直线构成；元素组成不同但线条属性明显；汉字、数字、字母可按曲直分类
- 题干触发词/图形特征：曲直性；全曲；全直；曲直混合；直线曲线
- 不适用情况：图形组成相同且只是方向变化时不看曲直；曲直数量变化复杂时转线数量卡
- 解题步骤：
  1. 判断整体是全曲、全直还是曲直混合。
  2. 若整体属性无规律，分别数直线数、曲线数或曲直分开看。
  3. 观察曲线/直线位置，如内外、上下、左右关系。
  4. 用选项匹配曲直属性或数量。
- 检查顺序：全曲/全直/曲直 -> 曲线数 -> 直线数 -> 曲直位置 -> 选项验证
- 考场速算/操作说明：curve_straight_screening，优先级 32。
- 常见错误：把弧线近似当直线；只看外轮廓不看内部线条；属性与数量混用
- 来源：花生图形推理笔记.pdf p.9,18,19,20；【判断推理】图形推理.pdf p.1

## 数量规律
### 一笔画与笔画数法（`gr_quantity_one_stroke_001`）
- 适用场景：出现五角星、圆相切/相交、田字变形、出头端点、T点、分离图形、特殊字符等；图形可视为线条网络；题干图形复杂但线条连接明显
- 题干触发词/图形特征：一笔画；奇点；笔画数；连通图
- 不适用情况：图形主要是面数量或元素数量规律时不要强行笔画；不连通图形需分部分计算，不能直接用全图奇点
- 解题步骤：
  1. 先看图形是否连通。
  2. 数奇点：通常0或2个奇点可一笔画。
  3. 多笔画时通常按奇点数/2并结合分离部分相加。
  4. 遇到出头点、T点、交点需严格区分。
  5. 选项必须满足同样笔画数或变化规律。
- 检查顺序：是否连通 -> 数奇点 -> 奇点/2 -> 分离部分相加 -> 选项验证
- 考场速算/操作说明：stroke_count，优先级 40。
- 常见错误：把端点当交点；忽略分离部分；多部分图形只看其中一部分
- 来源：花生图形推理笔记.pdf p.10,11；【判断推理】图形推理.pdf p.1

### 点数量与交点类型法（`gr_quantity_point_001`）
- 适用场景：线条相交较多；出现切点、圆相交、出头线、网格或交叉结构；题干强调点状关系
- 题干触发词/图形特征：点；交点；切点；出头点；曲直交点；内部交点
- 不适用情况：图形没有明显交叉或切点时不要优先数点；面或笔画提示更强时后置
- 解题步骤：
  1. 先确定要数哪类点：交点、切点、出头点、内部交点、外框交点等。
  2. 统一计数口径，不同图必须按同一类点计数。
  3. 注意出头点是端点，不是交点。
  4. 用数量恒定、递增递减、加减等关系验证选项。
- 检查顺序：选点类型 -> 统一口径 -> 逐图计数 -> 找规律 -> 选项验证
- 考场速算/操作说明：point_type_count，优先级 41。
- 常见错误：计数口径变化；出头点和交点混淆；漏数重叠处
- 来源：花生图形推理笔记.pdf p.11；【判断推理】图形推理.pdf p.1

### 线数量与线组关系法（`gr_quantity_line_001`）
- 适用场景：图形由明显线段或曲线组成；出现两个三角形、平行线、垂线等提示；属性看不出但线数量有变化
- 题干触发词/图形特征：线；直线；曲线；平行线组；线段数量；线位置
- 不适用情况：图形面数量提示更明显时先数面；线条有重合或分割不清时需标注不确定
- 解题步骤：
  1. 确定数直线、曲线、线段总数还是特殊线组。
  2. 必要时分开数横/竖/斜或平行线组。
  3. 观察线数是否恒定、递增递减、加减或分组。
  4. 选项线数和线组关系必须同规律。
- 检查顺序：数什么线 -> 是否分段 -> 是否分组 -> 是否平行/垂直 -> 选项验证
- 考场速算/操作说明：line_count_or_relation，优先级 42。
- 常见错误：把一条直线被交点分成多段后口径混乱；忽略内部线条；漏数重合线
- 来源：花生图形推理笔记.pdf p.9,12,15；【判断推理】图形推理.pdf p.1

### 角数量与角类型法（`gr_quantity_angle_001`）
- 适用场景：图形有明显折线或三角结构；出现直角、锐角、钝角可比较；角度呈递增递减
- 题干触发词/图形特征：角；直角；锐角；钝角；扇形中心角
- 不适用情况：大量交点但角定义不清时先谨慎；图形为封闭面且面数更明显时可后置
- 解题步骤：
  1. 先确定角的类型：总角、直角、锐角、钝角。
  2. 只数小于180度的角，且通常需由相交线段形成。
  3. 图形外部角一般不数，除非题干规律明确。
  4. 用选项验证角数或角度变化。
- 检查顺序：总角/直角/锐角/钝角 -> 计数口径 -> 逐图计数 -> 选项验证
- 考场速算/操作说明：angle_count，优先级 43。
- 常见错误：把180度直线当角；图形外角乱数；交点处多角漏数
- 来源：花生图形推理笔记.pdf p.12；【判断推理】图形推理.pdf p.1

### 封闭面数量与面属性法（`gr_quantity_face_001`）
- 适用场景：图形有明显封闭区域；出现分割图、图形内部格子或多个面；同时可数面/线/点时封闭面突出
- 题干触发词/图形特征：面；封闭空间；部分面；最大面；面形状；面积
- 不适用情况：图形没有封闭空间时不要数面；开闭性只是状态而非数量时不要混淆
- 解题步骤：
  1. 先数封闭面的总数。
  2. 若总数无规律，再看某一类面：三角形面、四边形面、多边形面或相同形状面。
  3. 单个大面可看最大面形状、面积、属性或与外框相似关系。
  4. 选项必须满足同一类面规则。
- 检查顺序：总面数 -> 某类面数 -> 最大/最小面 -> 面属性 -> 选项验证
- 考场速算/操作说明：face_count_then_face_property，优先级 44。
- 常见错误：把开放区域当面；重复数重叠区域；面数、线数、点数同时出现时选择混乱
- 来源：花生图形推理笔记.pdf p.13；【判断推理】图形推理.pdf p.1

### 元素种类与元素个数法（`gr_quantity_element_001`）
- 适用场景：出现多个小元素；生活化图形或黑色相连图形；不同元素种类或个数明显变化
- 题干触发词/图形特征：素；元素；种类；个数；生活化图形；黑色相连图形
- 不适用情况：元素之间有位置移动规律时不要只数个数；元素换算关系不清时不要编造等价
- 解题步骤：
  1. 先数元素种类，再数每种元素个数。
  2. 判断是否有元素间换算或等价关系。
  3. 若多个图形可分为数部分，先确定相连/分离规则。
  4. 用选项验证元素数量、种类或等价关系。
- 检查顺序：数种类 -> 数个数 -> 是否换算 -> 是否数部分 -> 选项验证
- 考场速算/操作说明：element_count_classification，优先级 45。
- 常见错误：把相连元素拆错；把外观相似但不同的元素归为一类；没有依据地设换算比例
- 来源：花生图形推理笔记.pdf p.14；【判断推理】图形推理.pdf p.1

## 数量/属性规律
### 图形间相离相交关系法（`gr_graph_relation_001`）
- 适用场景：题干由两个或多个图形组合；图形之间接触、相交、重叠明显；交点、公共线、重叠面变化
- 题干触发词/图形特征：图形间关系；相离；相交于点；相交于线；相交于面
- 不适用情况：单个图形内部规律明显时不优先图形间关系；组合关系不稳定且数量规律更清晰时后置
- 解题步骤：
  1. 判断图形之间是相离、相交于点、相交于线还是相交于面。
  2. 若相交于线，观察接触线数量、长短或位置。
  3. 若相交于面，观察重叠面的形状、属性或数量。
  4. 用选项验证组合关系是否一致。
- 检查顺序：相离 -> 交点 -> 交线 -> 交面 -> 形状/位置 -> 选项验证
- 考场速算/操作说明：inter_shape_relation，优先级 46。
- 常见错误：把接触点当重叠面；忽略公共边长短；只看单个图形不看组合关系
- 来源：花生图形推理笔记.pdf p.15；【判断推理】图形推理.pdf p.1

## 特殊题型
### 黑白块综合判断法（`gr_black_white_block_001`）
- 适用场景：图形主要由黑白块构成；黑白块在固定网格中排列；黑白块数量、连接或对称关系变化
- 题干触发词/图形特征：黑白块；面积；数量；连接；相邻比较；黑白运算
- 不适用情况：不是网格黑白块时不走此卡；格位相同且明显平移时先看位置平移
- 解题步骤：
  1. 先看对称性：整体或黑色/白色自身对称。
  2. 再看面积：黑色面积与整体面积、黑白面积关系。
  3. 再看数量：黑块数量是否相同，不同可看黑白差或直接数。
  4. 再看连接：组块个数、笔画数、连接方式。
  5. 最后进行相邻比较或黑白运算验证。
- 检查顺序：对称 -> 面积 -> 数量 -> 连接 -> 相邻比较/运算 -> 选项验证
- 考场速算/操作说明：black_white_multi_stage，优先级 18。
- 常见错误：黑数相同不先看平移；不按格位做比较；连接块数量和黑块数量混淆
- 来源：花生图形推理笔记.pdf p.17；【判断推理】图形推理.pdf p.1

### 汉字图形规律法（`gr_chinese_character_001`）
- 适用场景：题干为汉字或类汉字符号；汉字结构、偏旁、笔画、封闭面或读音特征明显
- 题干触发词/图形特征：汉字；结构；偏旁；笔画；封闭面；曲直；对称
- 不适用情况：汉字仅作为普通线条且有更明显图形规律时可回到通用卡；不得引入资料外的生僻汉字知识
- 解题步骤：
  1. 先看结构：内外、上下、左右。
  2. 再看样式：遍历、偏旁遍历、加减同异。
  3. 再看属性：对称性、曲直性、开闭性。
  4. 再看数量：点、线、面、部件数。
  5. 必要时看特殊：拼音音调、拼音字母个数、笔画顺序起笔。
- 检查顺序：结构 -> 样式 -> 属性 -> 数量 -> 特殊 -> 选项验证
- 考场速算/操作说明：symbol_as_shape_then_language_feature，优先级 19。
- 常见错误：只数笔画忽略结构；把汉字语义当规律；拼音类证据不足时乱用
- 来源：花生图形推理笔记.pdf p.18；【判断推理】图形推理.pdf p.1

### 数字图形规律法（`gr_digit_symbol_001`）
- 适用场景：题干主体为阿拉伯数字；数字形态或数字大小/运算可能成规律
- 题干触发词/图形特征：数字；对称性；曲直性；开闭性；封闭面；数字本身
- 不适用情况：数字只是标号或选项序号时不得纳入规律；图形规律已明显时不强行用数字本身运算
- 解题步骤：
  1. 先看图形属性：对称性、曲直性、开闭性。
  2. 再看封闭面数量。
  3. 再看数字本身：大小比较、排序或运算。
  4. 若出现数字组合，验证每一位或整体是否同规律。
- 检查顺序：对称 -> 曲直 -> 开闭 -> 封闭面 -> 数字本身 -> 选项验证
- 考场速算/操作说明：digit_shape_first_then_numeric，优先级 23。
- 常见错误：把题号当题干数字；数字本身运算与图形属性混淆；封闭面数量看漏8有两个面
- 来源：花生图形推理笔记.pdf p.19；【判断推理】图形推理.pdf p.1

### 字母图形规律法（`gr_letter_symbol_001`）
- 适用场景：题干主体为英文字母；字母形态或字母表顺序可能成规律
- 题干触发词/图形特征：字母；对称性；曲直性；开闭性；字母顺序；字母本身算法
- 不适用情况：字母只是选项标签时不得纳入规律；无字母本身规律证据时先按图形属性解
- 解题步骤：
  1. 先看字母图形属性：对称、曲直、开闭。
  2. 再看数量：面、点、线、素等。
  3. 最后看特殊：字母表顺序、字母本身算法。
  4. 选项需同时满足图形属性或字母序列规律。
- 检查顺序：对称 -> 曲直 -> 开闭 -> 数量 -> 字母表顺序/算法 -> 选项验证
- 考场速算/操作说明：letter_shape_first_then_alphabet，优先级 24。
- 常见错误：把选项A/B/C/D当题干字母；字母顺序规律证据不足时乱用；忽略大小写字形差异
- 来源：花生图形推理笔记.pdf p.20；【判断推理】图形推理.pdf p.1

## 空间重构
### 六面体展开图总流程（`gr_cube_net_overall_001`）
- 适用场景：题干为正方体/六面体展开图；选项为立体图或另一个展开图；需要判断能否折叠
- 题干触发词/图形特征：空间重构；六面体；展开图；还原母图；特殊展开图
- 不适用情况：平面图形规律题不使用空间重构；立体拼合题不要直接套相对面规则
- 解题步骤：
  1. 先识别展开图结构，如1-4-1、2-3-1、2-2-2、0-3-3等。
  2. 先用相对面排除：立体中相对面不能同时出现。
  3. 再用公共边或公共点验证相邻关系。
  4. 复杂选项用时针法描边验证方向。
  5. 特殊展开图原图不重要时，重点盯选项与公共边。
- 检查顺序：结构 -> 相对面 -> 公共边 -> 公共点 -> 时针法 -> 选项排除
- 考场速算/操作说明：cube_net_verification，优先级 10。
- 常见错误：相对面同时出现未排除；公共边方向看反；把展开图平面距离当立体相邻
- 来源：花生图形推理笔记.pdf p.21；【判断推理】图形推理.pdf p.1

### 相对面排除法（`gr_cube_opposite_faces_001`）
- 适用场景：正方体展开图中需要判断两个面能否相邻出现；选项立体同时展示若干面
- 题干触发词/图形特征：相对面；展开图；六面体；排除法
- 不适用情况：非六面体或非展开图题不用；两个面是否相对不确定时继续公共边验证
- 解题步骤：
  1. 在展开图中找相对面：同行/同列隔一面常为相对面；一列/一行两端也可能为相对面。
  2. 检查选项立体可见面中是否同时出现相对面。
  3. 若相对面同时出现，直接排除该选项。
  4. 剩余选项再做公共边或时针法验证。
- 检查顺序：标面 -> 找相对面 -> 排除同时出现 -> 再验公共边
- 考场速算/操作说明：opposite_face_elimination，优先级 11。
- 常见错误：把相邻面误判为相对面；只用相对面无法唯一确定却直接选
- 来源：花生图形推理笔记.pdf p.21；【判断推理】图形推理.pdf p.1

### 公共边验证法（`gr_cube_common_edge_001`）
- 适用场景：相对面排除后仍有多个候选；选项涉及两个或三个相邻面；面上有方向性图案
- 题干触发词/图形特征：公共边；展开图；相邻面；L型画边法
- 不适用情况：相对面已经能唯一排除时可不展开深入；公共边无法确定时使用公共点或时针法
- 解题步骤：
  1. 在平面图中构成直角的两条边通常是公共边。
  2. 一列/一行连着4个面时，最两侧的两条边可对应公共边。
  3. 3+1的L型同方向两条边可作为公共边。
  4. 沿公共边检查面上图案方向是否与选项一致。
- 检查顺序：找公共边 -> 标方向 -> 折叠对应 -> 选项比较
- 考场速算/操作说明：common_edge_trace，优先级 12。
- 常见错误：公共边找错导致方向反判；两个面的公共边唯一性忽略；只看面不看边
- 来源：花生图形推理笔记.pdf p.21；【判断推理】图形推理.pdf p.1

### 时针画边法（`gr_cube_clockwise_method_001`）
- 适用场景：展开图和选项均有方向性图案；公共边验证仍不足以排除；需要判断面内图案方向是否一致
- 题干触发词/图形特征：时针法；空间重构；方向验证；公共边
- 不适用情况：没有方向性标记时不优先使用；相对面已排除唯一选项时可不使用
- 解题步骤：
  1. 结合选项描画一个面。
  2. 找一个特殊面的唯一点或唯一边。
  3. 顺时针方向描述标号或图案顺序。
  4. 题干与选项对应面方向不一致则排除。
- 检查顺序：选特殊面 -> 定锚点/边 -> 顺时针读序 -> 选项比对
- 考场速算/操作说明：clockwise_orientation_check，优先级 13。
- 常见错误：起点不一致；顺逆方向混乱；面对应错位
- 来源：花生图形推理笔记.pdf p.21；【判断推理】图形推理.pdf p.1

### 立体截面图判断法（`gr_spatial_section_001`）
- 适用场景：题干给出立体图并问截面形状；出现六面体、圆柱、圆锥等常见立体；选项为平面截面图
- 题干触发词/图形特征：截面图；六面体；圆柱体；圆锥体；截面形状
- 不适用情况：展开图折叠题不使用截面法；复杂切割无法从资料确认时标记不确定
- 解题步骤：
  1. 六面体可截出三角形、四边形、五边形、六边形，注意不能截出直角梯形、正五边形等限制。
  2. 圆柱体可截出圆形、矩形、椭圆、抛物线与桶形等，注意资料标注无法截出梯形。
  3. 圆锥体可截出圆形、椭圆、抛物线、双曲线及三角形，注意资料标注无法截出四边形。
  4. 用“能否经过对应面/边/顶点”验证选项。
- 检查顺序：立体类型 -> 常见可截形状 -> 资料限制项 -> 选项排除
- 考场速算/操作说明：known_section_shape_filter，优先级 15。
- 常见错误：把圆锥斜截误判为四边形；忽略截面必须连续穿过立体；复杂选项硬猜
- 来源：花生图形推理笔记.pdf p.22；【判断推理】图形推理.pdf p.1

### 立体三视图还原法（`gr_spatial_three_view_001`）
- 适用场景：题干给出立体图或三视图；选项要求判断主视/左视/俯视；存在被遮挡部分
- 题干触发词/图形特征：三视图；主视图；左视图；俯视图；遮挡
- 不适用情况：平面规律题不使用；没有视角信息时不能强行推断
- 解题步骤：
  1. 观察到的视图都是平面图。
  2. 某些斜线或斜面可能被压平，注意透视压缩。
  3. 原图有线就有线，原图没线就没线。
  4. 被遮挡部分在对应视图中不可见。
  5. 用选项比对轮廓、线条和遮挡关系。
- 检查顺序：视角 -> 轮廓 -> 线条 -> 遮挡 -> 选项验证
- 考场速算/操作说明：orthographic_projection_check，优先级 16。
- 常见错误：把看不见的遮挡线画出来；把斜线透视关系看错；忽略视角方向
- 来源：花生图形推理笔记.pdf p.23；【判断推理】图形推理.pdf p.1

### 积木题分层画图法（`gr_spatial_assembly_001`）
- 适用场景：题干是多个正方体/积木拼合；要求判断缺块或能否拼成目标立体；选项为立体块组合
- 题干触发词/图形特征：立体拼合；积木题；分层画图；分层凑数
- 不适用情况：普通六面体折叠题不用；透视关系极不清晰时标记不确定
- 解题步骤：
  1. 先数个数。
  2. 试拼时优先找特殊块或“占地大”的块，最好横跨所有层级。
  3. 用分层画图法把立体转为平面层图。
  4. 必要时用分层凑数法想象0空间。
  5. 特殊块先拼，再用平面图确定空位。
- 检查顺序：数个数 -> 找特殊块 -> 分层画图 -> 分层凑数 -> 选项验证
- 考场速算/操作说明：layered_assembly，优先级 17。
- 常见错误：只看外观不数方块；上下层关系混乱；被遮挡方块漏数
- 来源：花生图形推理笔记.pdf p.24；【判断推理】图形推理.pdf p.1

## 选项验证
### 图推选项验证流程卡（`gr_option_verification_001`）
- 适用场景：已发现候选规律；选项中有多个看似相近答案；空间题、数量题或黑白运算题需要排除
- 题干触发词/图形特征：选项验证；排除法；反向验证；防猜
- 不适用情况：题干规律尚未形成时不要直接以选项倒推唯一答案
- 解题步骤：
  1. 先用题干已有图形验证候选规律至少两处。
  2. 再把每个选项代入空缺位置。
  3. 优先排除违反硬规则的选项，如相对面同时出现、格位运算不一致、数量口径不一致。
  4. 若多个选项满足，回到总流程寻找第二规律或更高优先级特征。
  5. 若规律不唯一，标记需要复核而不是强答。
- 检查顺序：题干验证 -> 选项代入 -> 硬规则排除 -> 多解回退 -> 输出答案
- 考场速算/操作说明：option_driven_validation_after_rule，优先级 8。
- 常见错误：只用一个局部特征排除；多个选项都满足时强行选；不同选项使用不同规则
- 来源：花生图形推理笔记.pdf p.3,4,21,22,24；【判断推理】图形推理.pdf p.1

## 易错点
### 图推常见误判点清单（`gr_common_misjudgment_001`）
- 适用场景：图推题出现多个可能规律；题干图形复杂或选项相似；AI解题器需要防止乱猜
- 题干触发词/图形特征：易错点；误判；防错；规则冲突
- 不适用情况：单一明确规律且选项唯一时只需简短复核
- 解题步骤：
  1. 先防组成误判：相同/相似/不同不要混。
  2. 防数量口径误判：点、线、角、面、素必须先定义计数对象。
  3. 防空间误判：相对面、公共边、时针法不能混用。
  4. 防黑白块误判：黑数相同时先看平移，格位一致才做运算。
  5. 防特殊符号误判：不要把选项标签当题干字母/数字。
- 检查顺序：组成判断 -> 计数口径 -> 空间关系 -> 黑白格位 -> 符号身份 -> 多解复核
- 考场速算/操作说明：anti_hallucination_checklist，优先级 9。
- 常见错误：为了匹配答案临时换规则；遇到难题强行输出确定答案
- 来源：花生图形推理笔记.pdf p.4,10,11,12,17,21,22,23,24；【判断推理】图形推理.pdf p.1

## 易错点

- 组成未判清就套规律。
- 点、线、角、面、素计数口径不统一。
- 黑白块格位不一致仍做黑白运算。
- 空间重构只看相似面，不做相对面/公共边/时针法验证。
- 汉字、数字、字母题把选项标签当题干信息。
