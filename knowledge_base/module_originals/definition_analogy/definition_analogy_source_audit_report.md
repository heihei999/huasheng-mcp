# 判断推理-定义判断与类比推理 source audit 报告（复审版）

## 本轮复审原因

用户指出上一版“判断推理-定义判断与类比推理”只纳入了《【判断推理】定义判断.pdf》和《【判断推理】类比推理.pdf》，遗漏了《花生判断笔记总结.pdf》。本轮重新整理时已将三份资料统一纳入来源审计。

## 纳入来源

- 【判断推理】定义判断.pdf：定义判断思维导图，核心覆盖总体思想、抓被定义词、主体/对象/特殊要求、属于/不属于、多定义枚举。
- 【判断推理】类比推理.pdf：类比推理思维导图，核心覆盖语义、语法、逻辑、对应、拆词关系。
- 花生判断笔记总结.pdf：第20页补充定义判断的选项设置、解题技巧、关键词查找技巧；第21-22页补充类比推理做题思想、三难点、逻辑关系、对应关系、语义二级辨析和语法结构。

## source_page 修正原则

- 定义判断卡片统一补充 `花生判断笔记总结.pdf` 第20页。
- 类比推理逻辑/对应关系卡片补充 `花生判断笔记总结.pdf` 第21页。
- 类比推理语义/语法/拆词卡片补充 `花生判断笔记总结.pdf` 第22页。
- 类比推理总流程、二级辨析、防错卡补充第21-22页。
